# Security Policy

## Supported Versions
This project does not currently publish supported release branches. Please use the latest commit on the default branch.

## Reporting a Vulnerability
If you discover a security issue, please report it privately whenever possible.

Preferred:
- Open a GitHub Security Advisory for this repository.

Alternative:
- Open a GitHub issue with minimal details and request a private channel for follow-up.

We will acknowledge reports as soon as possible and coordinate a fix and disclosure plan.
